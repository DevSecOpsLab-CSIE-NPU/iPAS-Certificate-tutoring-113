## 安裝 Numpy 套件

要開始使用 NumPy，首先需要安裝它。以下是詳細的安裝步驟：

### 使用 pip 安裝 📦

1. 打開終端或命令提示字元。
2. 輸入以下命令來安裝 NumPy：
    ```bash
    pip install numpy
    ```
3. 等待安裝完成。你會看到類似以下的訊息：
    ```
    Successfully installed numpy
    ```

    ```

### 驗證安裝 ✅

1. 打開 Python 互動式環境（例如，輸入 `python` 或 `ipython`）。
2. 輸入以下命令來驗證 NumPy 是否安裝成功：
    ```python
    import numpy as np
    print(np.__version__)
    ```
3. 如果沒有錯誤訊息並且顯示 NumPy 的版本號，表示安裝成功！🎉
